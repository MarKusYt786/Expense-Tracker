# 💰 Expense Tracker

A modern Expense Tracker built with React that helps users manage income and expenses through a clean dashboard with authentication, charts, filters, and financial insights.

## 🚀 Live Demo

https://expense-tracker-xi-five-63.vercel.app/

## 📂 GitHub Repository

https://github.com/MarKusYt786/expense-tracker

---

## ✨ Features

- User Registration & Login
- Protected Routes
- Persistent Authentication
- User Profile Avatar
- Add Income & Expenses
- Edit & Delete Transactions
- Filter Transactions
- Financial Summary
- Monthly Expense Chart
- Income & Expense Overview
- Top Spending Categories
- Financial Insights
- Responsive Design
- Local Storage Persistence
- Per-user Transaction Storage
- Toast Notifications
